# Python Beginner Exercises

This repository contains a collection of beginner-friendly Python exercises that I completed to practice and strengthen my programming skills.

## Purpose

The goal of this project is to learn and apply the fundamental concepts of Python programming through simple, practical exercises. Each exercise focuses on one or more core programming concepts such as:

- Variables
- User input and output
- Arithmetic operations
- Conditional statements (`if`, `elif`, `else`)
- Loops (`for` and `while`)
- String manipulation
- Basic program logic

## Repository Structure

### `instructions.txt`
Contains the descriptions and objectives for each exercise, including the expected output and learning goals.

### `solutions.py`
Contains my Python solutions for all exercises.

## Exercises Included

1. Greeting Program
2. Simple Calculator
3. Even or Odd Checker
4. Age Checker
5. Count From 1 to 10
6. Multiplication Table
7. Number Guessing Game
8. Word Counter
9. Shopping Cart Total
10. Mini Password Checker
11. Student Grade Calculator

## What I Learned

By completing these exercises, I practiced:

- Receiving and validating user input
- Performing mathematical calculations
- Making decisions using conditional statements
- Repeating actions with loops
- Working with strings and text processing
- Organizing code and solving problems step by step

## How to Run

1. Clone this repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project folder:
   ```bash
   cd <repository-name>
   ```

3. Run the Python file:
   ```bash
   python solutions.py
   ```
